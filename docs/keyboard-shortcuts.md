# Keyboard Shortcuts

All keyboard shortcuts available in cmux.

## Workspaces

| Action | Shortcut |
|--------|----------|
| New workspace | `Ctrl+Alt+\` |
| Next workspace | `Ctrl+Tab` |
| Previous workspace | `Ctrl+Shift+Tab` |
| Close current workspace/file | `Ctrl+W` |
| Rename workspace | `F2` |

## General

| Action | Shortcut |
|--------|----------|
| Toggle Scratch Pad | `Ctrl+J` |
| Show keyboard shortcuts help | `Ctrl+Shift+?` |
